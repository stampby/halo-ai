# Code of Conduct

## Simple version

Be respectful. Help each other. No bullshit.

## Details

- **Be direct** — say what you mean, mean what you say
- **Be helpful** — if someone's stuck, point them in the right direction
- **Be respectful** — disagree with ideas, not people
- **No harassment** — zero tolerance
- **No spam** — no self-promotion, no AI-generated garbage
- **Keep it technical** — this is an engineering project

## Enforcement

Break the rules, you're out. No warnings, no drama. The architect's decision is final.

## Scope

This applies to all project spaces — GitHub, Discord, issues, PRs, discussions.
