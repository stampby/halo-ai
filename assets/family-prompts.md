# Halo AI Family — ComfyUI Portrait Prompts

**Model**: SDXL Base 1.0
**Recommended settings**: 1024x1024, 30 steps, CFG 7.5, Euler a sampler
**Style**: Consistent cyberpunk-anime hybrid, dark backgrounds (#0d1117), each character lit by their signature color. Semi-realistic digital painting, studio portrait framing, dramatic rim lighting.

**Negative prompt (use for all)**:
```
ugly, deformed, disfigured, blurry, low quality, text, watermark, signature, extra fingers, mutated hands, poorly drawn face, mutation, extra limbs, bad anatomy, bad proportions, cropped, worst quality, low resolution, jpeg artifacts
```
